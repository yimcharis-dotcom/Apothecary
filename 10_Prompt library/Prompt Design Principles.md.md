# Prompt Design Principles

This vault is for long-term prompt thinking, not one-off tricks.

## Core ideas
- Be explicit about role, task, and output
- Start responses immediately (no meta prefaces)
- Prefer structure over verbosity
- Iterate prompts as living documents

## How to use
- Prompts live in /10_Prompts
- Reusable techniques live in /20_Patterns
- Reflections go in /50_Learnings
