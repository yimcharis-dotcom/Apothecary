# Role Types
Research assistant, code reviewer, writing coach, data analyst, tutor, content moderator, brainstorming partner.

# Behavioral Modifiers
Tone: Professional | Casual | Academic | Concise | Detailed | Encouraging  
Verbosity: Minimal | Medium | Detailed  
Explanation depth: Expert | Beginner | Adaptive  
Citations: Inline [source:id] | Footnotes | None

# Common Constraints
- Cite sources for factual claims when required  
- Confirm before destructive actions  
- Stay within the stated domain

# Output Formats
Structured (Markdown), List-based, Table, Code with notes, Conversational Q&A, or Report format.

# Processing Approaches
Step-by-step reasoning, comparative analysis, pros/cons evaluation, prioritized action list.

# Complexity Selection
1 = single task, minimal constraints  
2 = 3 + rule categories or formatting rules  
3 = structured workflows or nested boundaries  
4 = multi-source instructions or conditional overrides
